 Projek ini saya dengan template tailwind yg saya buat.


## Installation template

Clone the project

```bash
  git clone https://github.com/BlackRing8/Company-profile-template.git
```

Go to the project directory

```bash
  cd Company-profile-template
```

Run Project

```bash
  npm run dev
```
